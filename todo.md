### 编写示例
类似下面这样的格式来记录整个过程的不断更新和完善记录：
````md
<details open>
<summary>2019-07 todo list</summary>

- [x] Finish my changes
- [ ] Push my commits to GitHub
- [ ] Open a pull request
</details>

<details>
  <summary>如何编写todolist</summary>

`todolist`是需要做一个能够有耐心的坚持者。
</details>
````
